What We Built

4 AI-Powered Education Tools

Each project is a fully independent application with its own UI theme, local persistence via IndexedDB, and AI-driven features powered by Z.ai.

CodeTutor

Socratic Coding Mentor

AI Tutor

An AI coding tutor that guides learners through problems using the Socratic method — never giving answers directly, but leading you to discover solutions yourself. Features Monaco Editor, 3-level progressive hints, and solution comparison.

Monaco Editor Progressive Hints 6 Languages Stats Dashboard
View on GitHub

SmartFlashcard

AI Flashcard Generator

Spaced Repetition

Upload PDFs or text, and AI generates structured flashcards with difficulty grading. Study with the SM-2 spaced repetition algorithm for optimized long-term retention. Features 3D flip animations, deck management, and Anki-compatible export.

SM-2 Algorithm 3D Flip Cards PDF Upload CSV Export
View on GitHub

PaperChat

Document Q&A Assistant

RAG

Upload research papers or documents and have intelligent conversations with their content. Uses lightweight RAG with text chunking and keyword matching for citation-backed answers, auto-summaries, and suggested follow-up questions.

Citation Tracking Auto Summary Document Outline Chat Export
View on GitHub

SyllabusRoadmap

Visual Learning Path Generator

React Flow

Paste any course syllabus and AI transforms it into an interactive DAG visualization using React Flow. Track progress by clicking nodes, switch between layouts and color themes, and export your roadmap as a PNG image.

DAG Visualization Progress Tracking 5 Color Themes PNG Export

Built Client-Side Only

No backend. No server. Pure browser-native AI applications.

architecture.ts 
const stack = {
  framework:    "React 19" + "TypeScript 5.9",
  bundler:      "Vite 8",
  styling:      "Tailwind CSS v4",
  state:        "Zustand",
  persistence:  "IndexedDB" // via idb library,
  ai:           "OpenAI SDK" // Z.ai compatible,
  animations:   "Framer Motion",
};

const features = [
  "No backend required",
